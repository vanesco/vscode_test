시큐리티 솔루션

보메트릭은 엔터프라이즈 암호화 및 키 관리 분야의 전문 솔루션으로, VDS(Vormetric Data Security) 솔루션은
탑재 장소에 관계없이 어떤 유형의 파일, 데이터베이스, 어플리케이션도 성능 저하나 키 관리 복잡성 없이 암호화할 수 있도록
해주는 확장성 있고 관리가 쉬운 단일 솔루션을 제공합니다.
IBM은 DB2 및 Informix를 위한 유일한 데이터베이스 암호화 솔루션으로 보메트릭 기술을 채택하였으며,
Symantec 또한 Veritas NetBackup Media Server 암호화 옵션 제공을 위해 보메트릭 기술을 사용하고 있습니다.

보메트릭의 암호화 대상은 IIS, Apache, WebLogic 과 같은 미들웨어에서 부터 각종 DBMS와 다양한 어플리케이션, File server,
FTP server 등의 데이터와 백업 및 설정정보까지 저장되는 모든 데이터에 암호화를 적용 할 수 있습니다.

  • Data Integrator
  • 도입의 필요성

    - DB 암호화에 따른 개인정보 보호법
    - DB 암호화 시 복잡한 구현 방식 및 구축후 성능 저하 이슈
    - DB 암호화 구축 시 연동되는 어플리케이션 소스 수정 문제
    - DB 자체 암호화 시 로그파일 등의 개인정보 유출 우려 이슈


    특장점

    - 손쉬운 설치 및 간단한 로직구현으로 인한 아키텍처 유연성
    - 어플리케이션의 소스 수정이 필요없는 암호화 환경 구축
    - 성능 저하없는 DB암호화 구축
    - 높은 호환성과 보안성
